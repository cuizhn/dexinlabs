# 新增 Architecture Principle 8

请将以下内容加入 ARCHITECTURE.md：

---

## Principle 8

### Single Responsibility Ownership（唯一职责归属）

每一种能力，在整个项目中只能有一个真正负责它的模块。

例如：

Markdown：

```
markdown-engine
```

内容：

```
content-engine
```

数据库：

```
database
```

对象存储：

```
storage
```

页面、组件、Query、API 不允许重复承担这些职责。

禁止多个模块共同维护同一种能力。

所有能力必须拥有唯一归属。
