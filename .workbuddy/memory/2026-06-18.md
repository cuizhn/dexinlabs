# 2026-06-18 工作日志

## 代码检查与优化（dexinlabs）

全量审阅项目代码，修复以下问题：

### Bug 修复
1. **MarkdownRenderer.vue 严重 Bug**：组件内部重复 useAsyncData 查询文档（父组件已传入 :value prop），导致冗余请求；props 定义顺序错误；模板使用内部 chapterDoc 而非 prop；watch 监听的 prop 名与定义不符。已重写为正确使用父组件传入的 value prop。
2. **首页标题双重拼接**：pages/index.vue 的 useHead title 为 `'Edu Platform - 用数学理解世界'`，app.vue 已有 titleTemplate `'%s - Edu Platform'`，会导致 `'Edu Platform - 用数学理解世界 - Edu Platform'`。已修正为 `'用数学理解世界'`。
3. **AppHeader useRoute 位置错误**：currentPath 计算属性内调用 useRoute()，应在 setup 顶层调用。已修正。
4. **[chapter].vue 缺少 @rendered 事件处理**：MarkdownRenderer 发出 rendered 事件传递 toc 数据，但父页面未监听，导致 tocItems 始终为空、右侧目录栏永远不显示。已补充 `@rendered="({ toc }) => tocItems = toc"`。

- `pages/courses/[slug]/index.vue`：移除冗余 `import { useRoute } from 'vue-router'`（Nuxt 自动注入）
