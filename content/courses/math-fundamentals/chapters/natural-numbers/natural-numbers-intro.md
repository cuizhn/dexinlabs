# 自然数的概念

## 什么是自然数

自然数是最基本的数，用于表示物体的个数。

> 定义：自然数是从 0 或 1 开始的正整数序列。

不同的定义体系中，自然数的起点可能不同：
- **集合论定义**：自然数从 0 开始
- **传统数学定义**：自然数从 1 开始

在本课程中，我们采用集合论定义，即自然数包括 0。

## 自然数的表示

自然数可以用多种方式表示：

### 数字表示
$$0, 1, 2, 3, 4, 5, \dots$$

### 集合表示
$$\mathbb{N} = \{0, 1, 2, 3, \dots\}$$

## 自然数的性质

1. **有序性**：自然数可以按照大小顺序排列
2. **无限性**：自然数的个数是无限的
3. **最小性**：存在最小的自然数 0
4. **封闭性**：自然数的加法和乘法运算结果仍是自然数
