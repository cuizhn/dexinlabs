# 整数与负数

## 为什么需要负数

在日常生活中，我们经常遇到相反意义的量：

- 温度：零上 5°C 和零下 5°C
- 收支：收入 100 元和支出 100 元
- 位置：向东走 5 米和向西走 5 米

为了表示这些相反意义的量，我们引入了负数。

## 整数的定义

整数包括正整数、零和负整数：

$$\mathbb{Z} = \{\dots, -3, -2, -1, 0, 1, 2, 3, \dots\}$$

## 数轴

数轴是表示数的直线：

```
<--|-----|-----|-----|-----|-----|-----|-----|-----|-->
  -3    -2    -1     0     1     2     3     4
```

### 数轴的特点

1. **原点**：表示 0 的点
2. **正方向**：通常向右为正方向
3. **单位长度**：相邻两个整数之间的距离相等

## 整数的运算

### 加法
$$a + (-b) = a - b$$
$$-a + (-b) = -(a + b)$$

### 减法
$$a - (-b) = a + b$$

### 乘法
$$(-a) \times (-b) = ab$$
$$(-a) \times b = -ab$$
